# AI Character Consultant Portfolio

Professional portfolio website showcasing AI character creation expertise and social media growth results.

## 🚀 Quick Deploy

### Option 1: Deploy to Vercel (Recommended)

1. **Visit**: [https://vercel.com](https://vercel.com)
2. **Sign up/Login** with your GitHub, GitLab, or Bitbucket account
3. Click **"Add New Project"**
4. **Import your Git repository** or drag and drop your project folder
5. Vercel will auto-detect the settings
6. Click **"Deploy"**
7. Your site will be live in ~1 minute at a free `.vercel.app` URL

**Custom Domain (Optional)**: You can add your own domain in Vercel settings for free!

### Option 2: Deploy to Netlify

1. **Visit**: [https://netlify.com](https://netlify.com)
2. **Sign up/Login** with your GitHub, GitLab, or Bitbucket account
3. Click **"Add new site"** → **"Deploy manually"**
4. Drag and drop the **`dist`** folder (after running `npm run build`)
5. Your site will be live at a free `.netlify.app` URL

**OR** use Netlify CLI:
```bash
npm install -g netlify-cli
npm run build
netlify deploy --prod
```

### Option 3: Deploy to GitHub Pages

1. Create a new GitHub repository
2. Push your code to GitHub
3. Go to repository Settings → Pages
4. Set source to "GitHub Actions"
5. Create `.github/workflows/deploy.yml` (file included)
6. Push changes - your site will auto-deploy!

## 📝 Customization

To update your contact information:
1. Open `src/App.tsx`
2. Find the `#contact` section
3. Update the email link: Change `mailto:your.email@example.com` to your actual email
4. Update LinkedIn URL if needed
5. Rebuild: `npm run build`
