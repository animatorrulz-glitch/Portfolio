# ⚡ Quick Start - Deploy in 2 Minutes!

## Fastest Way to Deploy (Vercel)

### Step 1: Build Your Portfolio
```bash
npm install
npm run build
```

### Step 2: Deploy to Vercel

**Option A - Using Vercel Website (No coding needed)**:

1. Go to: **https://vercel.com/signup**
2. Sign up with GitHub (or any method)
3. After login, click **"Add New..."** → **"Project"**
4. Choose **"Deploy from existing repository"** or **"Upload"**
5. If uploading: Drag the entire project folder
6. Click **"Deploy"**
7. ✅ **DONE!** Your portfolio is live!

**Option B - Using Command Line**:

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy (run in your project folder)
vercel

# Follow prompts, press Enter to accept defaults
# For production deployment:
vercel --prod
```

### Step 3: Share Your Portfolio! 🎉

Your portfolio will be live at: `https://your-project.vercel.app`

## Before You Share

### Update Your Email:

1. Open `src/App.tsx` in any text editor
2. Find line with `mailto:your.email@example.com`
3. Replace with your real email address
4. Save the file
5. Redeploy:
   ```bash
   npm run build
   vercel --prod
   ```
