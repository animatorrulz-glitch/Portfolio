📱 EASIEST METHOD - VERCEL (RECOMMENDED)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://vercel.com
2. Click "Sign Up" (use email or GitHub)
3. Click "Add New..." → "Project"
4. Upload your entire project folder
5. Click "Deploy"
6. Wait 1 minute
7. ✅ DONE! Copy your URL!

Your URL: https://your-name.vercel.app


💻 COMMAND LINE METHOD (FASTEST)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WINDOWS:
--------
1. Press Windows + R
2. Type: cmd
3. Type: cd C:\path\to\your\portfolio
4. Type: npm install -g vercel
5. Type: vercel login (browser opens, click confirm)
6. Type: vercel --prod
7. ✅ DONE! Copy your URL!

MAC:
----
1. Press Cmd + Space
2. Type: terminal
3. Type: cd /path/to/your/portfolio
4. Type: npm install -g vercel
5. Type: vercel login (browser opens, click confirm)
6. Type: vercel --prod
7. ✅ DONE! Copy your URL!


🔷 ALTERNATIVE - NETLIFY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Run: npm run build (in command line)
2. Go to: https://netlify.com
3. Click "Sign Up"
4. Drag the "dist" folder to Netlify
5. ✅ DONE! Copy your URL!
